# 🧭 Room Navigator

A Progressive Web App (PWA) designed to help freshmen and visitors easily understand NEU’s room codes.  
By simply entering a code (e.g., `M401B`) into the search bar, the app instantly translates it into clear building, floor, and room information.  
A built‑in legend guide provides quick reference for building abbreviations, making navigation simple and stress‑free.

🔗 **Live deployed link:** [room-navigator.vercel.app](https://room-navigator.vercel.app/)

---

## 📖 Project Statement
Room Navigator focuses on clarity, accessibility, and ease of use — ensuring that new students can confidently find their way around campus without confusion.  
It reduces anxiety during the first weeks of school by translating confusing codes into plain language.

---

## ✨ Features
- **Room Code Translation** — enter a code like `M401B` and instantly see building name, floor, and room number.  
- **Legend Guide** — quick reference for building abbreviations (M = Main Building, SOM = School of Management, PSB = Professional Building, IS = Integrated School).  
- **Simple Interface** — clean search bar with instant results for freshmen navigation.  
- **Progressive Web App (PWA)** — installable on mobile devices for native‑like experience.  
- **Fast Deployment** — lightweight front‑end hosted on Vercel for easy access.  

---

## 🔄 Application Flows
| Flow | Purpose | Key Steps |
|------|----------|-----------|
| **1. Room Code Translation (Search Bar)** | Core feature for freshmen navigation | - User enters a room code in the search bar (e.g., `M401B`)<br>- System parses building abbreviation, floor, and room number<br>- Display clear output: *Main Building, 4th Floor, Room 401* |
| **2. Legend Guide** | Helps users understand abbreviations | - Provide quick reference for building codes<br>- Examples: M = Main Building, SOM = School of Management, PSB = Professional Building, IS = Integrated School |
